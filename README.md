# CCNA-Labs

This repository contains my hands-on networking labs completed while studying CCNA, CCNP Enterprise, and Network Security.

## Topics Covered

* Network Fundamentals
* IPv4 & IPv6
* VLANs
* Inter-VLAN Routing
* Spanning Tree Protocol (STP)
* EtherChannel
* Static Routing
* OSPF
* EIGRP
* ACLs
* NAT
* DHCP
* DNS
* SSH
* Network Security Fundamentals

## Lab Environment

* Cisco Packet Tracer
* Cisco IOS
* Wireshark (future)
* Palo Alto (future)
* FortiGate (future)
