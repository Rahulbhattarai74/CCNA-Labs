# CCNA-Labs

This repository contains my hands-on networking labs completed while studying CCNA, CCNP Enterprise, and Network Security.

## Topics Covered

- Network Fundamentals
- IPv4 & IPv6

- VLANs
  - DTP & VTP
  - Legacy Inter-VLAN Routing
  - Router-on-a-Stick (ROAS)
  - Switch Virtual Interfaces (SVI)

- Switching
  - Spanning Tree Protocol (STP)
  - EtherChannel

- Routing
  - Static Routing
  - OSPF
  - EIGRP

- Network Services
  - DHCP
  - DNS
  - NAT

- Security & Management
  - ACLs
  - SSH
  - Network Security Fundamentals

## Lab Environment

* Cisco Packet Tracer
* Cisco IOS
* Wireshark (future)
* Palo Alto (future)
* FortiGate (future)
